Drew'sFlower Shop

Shopify flower store

Visit the site Store password: devdrew GitHub Repo

Drew's Flower Shop is a shopify store based on the website fromyouflowers.com.

This is my first solo project and thus I tried to incorporate as many shopify features as I could think of. Mostly adapting features from Shopify's Dawn theme to replicate features of the fromyouflowers website.

I would say the part I'm proud of the most is the functionality of the image gallery on the product page. See assets/carousel.js in the GitHub repo

That's because most of it is my original code and the only guidance I could find for it were the vague suggestions of how the code works that I gathered from the Firefox dev tools.
