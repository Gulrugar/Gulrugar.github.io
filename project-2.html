Drew'sYoutube Merch

Youtuber Branded Merchandise

Visit the site Store password: devdrew GitHub Repo

Drew's Youtube Merch is a shopify store based on the website merch.gamegrumps.com.

This project started out with me wanting to learn how to make certain features animate based on the window resizing and scrolling.

But as I made those features I also found even more eye-catching effects and ended up remaking the whole site.

When viewing the project I recommend:

  1. Resizing the window width and watch the header
  2. Scrolling down the homepage on a fullscreen and watching the
    left and right margins
